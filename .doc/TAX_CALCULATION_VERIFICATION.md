## 计算验证

### 1月验证
```
本年应税收入合计：65,119.35
年度税额 = 65,119.35 × 0.10 - 2,520.00 = 3,991.94 ✅
当月预扣税 = 3,991.94 - 0 = 3,991.94 ✅
```

### 2月验证
```
本年应税收入合计：81,813.96
年度税额 = 81,813.96 × 0.10 - 2,520.00 = 5,661.40 ✅
当月预扣税 = 5,661.40 - 3,991.94 = 1,669.46 ✅
```

### 3月验证
```
本年应税收入合计：98,867.12
年度税额 = 98,867.12 × 0.10 - 2,520.00 = 7,366.71 ✅
当月预扣税 = 7,366.71 - 5,661.40 = 1,705.31 ✅
```

### 4月验证
```
本年应税收入合计：140,910.57
年度税额 = 140,910.57 × 0.10 - 2,520.00 = 11,571.06 ✅
当月预扣税 = 11,571.06 - 7,366.71 = 4,204.35 ✅
```

### 5月验证 ✅
```
本年应税收入合计：174,357.12
年度税额 = 174,357.12 × 0.20 - 16,920.00 = 17,951.42 ✅
当月预扣税 = 17,951.42 - 11,571.06 = 6,380.36 ✅

关键点：
- 累计应税收入174,357.12超过了144,000元，税率从10%跳至20%
- 速算扣除数从2,520变为16,920
- 往月预扣税合计：11,571.06（1-4月累计）
```

**注意**：5月税率从10%变为20%，因为累计应税收入超过了144,000元（进入20%税率档）

### 年终奖验证（4月）

年度奖金计税额：190,000.00

适用税率：20%（月均额在12,000-25,000区间）

## 税率档次说明

### 年度综合所得税率表（累计预扣法使用）
- **不超过36,000元**：3%，速算扣除数 0
- **超过36,000至144,000元**：10%，速算扣除数 2,520
- **超过144,000至300,000元**：20%，速算扣除数 16,920 ⬅️ 5月进入此档
- **超过300,000至420,000元**：25%，速算扣除数 31,920

## 关键发现

1. **5月税率跳档**：累计应税收入174,357.12超过了144,000元，税率从10%跳至20%
2. **累计预扣法**：每月税额 = 累计应缴税额 - 往月预扣税合计
3. **年终奖单独计税**：使用月度换算税率表，不参与累计预扣
