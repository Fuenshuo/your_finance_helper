# Flux Ledger - Clean Architecture

## Overview

Flux Ledger is a streamlined personal finance application following Elon Musk's First Principles methodology. The codebase has been cleaned to focus exclusively on core Flux functionality.

## Core Components

- **main_flux.dart**: Application entry point
- **unified_transaction_entry_screen.dart**: AI-powered transaction input
- **flux_router.dart**: Navigation system
- **flux_providers.dart**: State management
